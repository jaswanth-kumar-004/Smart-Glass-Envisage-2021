# Smart-Glass-Envisage-2021

Features
-----------

- Date (Page 1)
- Time (Page 1)
- Battery Percentage (Page 1)
- No. of Steps (Page 2)
- Humidity (Page 3)
- Temperature (Page 3)
- Caller Notification
- Social Distancing Alert
- Button to switch pages

Components Used
----------------

- Arduino Mega 2560
- OLED 128x64 Display
- DHT11 Sensor
- PIR Sensor
- Inbuilt Pedometer in Phone
- HC-05 Bluetooth Module
